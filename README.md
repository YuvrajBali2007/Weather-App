# Weather App

A simple weather application built using HTML, CSS, and JavaScript.

## Features

- Search weather by city
- Live weather using OpenWeather API
- Displays:
  - City
  - Temperature
  - Humidity
  - Wind Speed

## Technologies Used

- HTML
- CSS
- JavaScript
- OpenWeather API

## How to Run

1. Download the project.
2. Open `weather.html` in your browser.
3. Enter a city name and click **Search**.
